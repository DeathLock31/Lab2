## Задание 1
### 1-1 Постановка задачи

> На вход программы подаются целое число a и натуральное число n.
> Необходимо вывести через пробел n чисел последовательности (начиная со второго элемента), заданной по следующим правилам:
> - Первый элемент последовательности - это число a.
> - Каждый последующий член последовательности - это умноженный на 2 остаток предыдущего члена последовательности от деления на 10.

### 1-2. Входные и выходные данные

#### Данные на вход

На вход программа должна получать целое число a и натуральное число n. В условии нет ограничений на максимальное значение данных на вход чисел, поэтому будем считать, что число a может принимать любые значения, входящие в диапазон типа данных long, а число n может принимать значения от 1 до максимального значения типа данных long.

|   | Тип                | min значение     | max значение       | 
|---|--------------------|------------------|--------------------|
| a | Целое число (long) | -2<sup>63</sup>  | 2<sup>63</sup> - 1 |
| n | Целое число (long) | 1                | 2<sup>63</sup> - 1 |

#### Данные на выход

По условию задачи, на выход мы получаем n целых чисел, разделённых через пробел.

### 1-3. Выбор структуры данных

Программа получает целое и натуральное числа, поэтому для их хранения можно выделить 2 переменных (`a` и `n`) типа `long`.

|   | название переменной | Тип (в Java)| 
|---|---------------------|-------------|
| a | `a`                 | `long`      |
| n | `n`                 | `long`      | 


Вывод результата - это целые числа. Каждое будет хранится перед выводом в 1 переменной (`t`) типа `long`.
|   | название переменной | Тип (в Java)| 
|---|---------------------|-------------|
| t | `t`                 | `long`      |

### 1-4. Алгоритм

#### Алгоритм выполнения программы:

1. **Ввод данных и создание переменной для вывода результата**  
   Программа считывает два числа, обозначенные как `a` и `n`. Затем создаёт переменную `t` и присваивает ей значение, хранящееся в переменной `a`.

2. **Вход в цикл**
   Начинается цикл со счётчиком for, который произойдёт n раз.

3. **Присвоение нового значения переменной t и вывод части результата**
   Значение `t` заменяется на остаток от деления `t` на 10. Затем происходит вывод значения `t` и пробел.

4. **Повторение цикла**
   Далее цикл повторяется n раз.



### 1-5. Программа

```java
import java.io.PrintStream;
import java.util.Scanner;

public class Main {
    // Объявляем объект класса Scanner для ввода данных
    public static Scanner in = new Scanner(System.in);
    // Объявляем объект класса PrintStream для вывода данных
    public static PrintStream out = System.out;

    public static void main(String[] args) {
        // Считываем значения трёх целочисленных значений из консоли
        long A = in.nextLong();
        long B = in.nextLong();
        long C = in.nextLong();

        // Проверяем, есть ли нуль среди введённых чисел и выводим следующее сообщение, если есть
        if (A*B*C == 0) {
            out.println("Есть нулевые");
        }
        else {
            // Если нуля среди чисел нет, то проверяем, все ли числа положительные,
            // и выводим следующее сообщение, если это так
            if (A > 0 && B > 0 && C > 0) {
                out.println("Все числа положительные");
            }
            else {
                // Если не все числа положительные, проверяем являются ли все числа отрицательными,
                // и выводим следующее сообщение, если это так
                if (A < 0 && B < 0 && C < 0) {
                    out.println("Все числа отрицательные");
                }
                else {
                    // Если ни одно условие выполнено не было, то выводится следующее сообщение
                    out.println("Все числа положительные или отрицательные");
                }
            }
        }

    }
}
```
