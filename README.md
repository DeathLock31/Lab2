## Задание 1
### 1-1 Постановка задачи

> На вход программы подаются целое число a и натуральное число n.
> Необходимо вывести через пробел n чисел последовательности (начиная со второго элемента), заданной по следующим правилам:
> - Первый элемент последовательности - это число a.
> - Каждый последующий член последовательности - это умноженный на 2 модуль остатка предыдущего члена последовательности от деления на 10.

### 1-2. Входные и выходные данные

#### Данные на вход

На вход программа должна получать целое число a и натуральное число n. В условии нет ограничений на максимальное значение данных на вход чисел, поэтому будем считать, что число a может принимать любые значения, входящие в диапазон типа данных long, а число n может принимать значения от 1 до максимального значения типа данных long.

|   | Тип                | min значение     | max значение       | 
|---|--------------------|------------------|--------------------|
| a | Целое число (long) | -2<sup>63</sup>  | 2<sup>63</sup> - 1 |
| n | Целое число (long) | 1                | 2<sup>63</sup> - 1 |

#### Данные на выход

По условию задачи, на выход мы получаем n целых чисел, разделённых через пробел.

### 1-3. Выбор структуры данных

Программа получает целое и натуральное числа, поэтому для их хранения можно выделить 2 переменных (`a` и `n`) типа `long`.

| n | `n`                 | `long`      | 


Вывод результата - это целые числа. Каждое будет хранится перед выводом в 1 переменной (`t`) типа `long`.
|   | название переменной | Тип (в Java)| 
|---|---------------------|-------------|
| t | `t`                 | `long`      |

### 1-4. Алгоритм

#### Алгоритм выполнения программы:

1. **Ввод данных и создание переменной для вывода результата**   
       Программа считывает два числа, обозначенные как `a` и `n`. Затем создаёт переменную `t` и присваивает ей значение, хранящееся в переменной `a`.

2. **Вход в цикл**   
       Начинается цикл со счётчиком for, который произойдёт n раз.

3. **Присвоение нового значения переменной t и вывод части результата**   
       Значение `t` заменяется на умноженный на 2 модуль остатка от деления `t` на 10. Затем происходит вывод значения `t` и пробел.

4. **Повторение цикла**   
       Далее цикл повторяется n раз.



### 1-5. Программа

```java
import java.io.PrintStream;
import java.util.Scanner;

public class Main {
    public static PrintStream out = System.out;
    public static Scanner in = new Scanner(System.in);
    public static void main(String[] args) {
        long a = in.nextInt(), n = in.nextInt();
        long t = a;

        for (int i = 0 ; i < n ; i++) {
            t = Math.abs(t % 10 * 2);
            out.print(t + " ");
        }
    }
}
```

## Задание 2
### 2-1 Постановка задачи

> Дана послоедовательность из n чисел. Найти такой индекс, где максимум последовательности встречается k-ый раз. Если такого индекса нет - вывести "NO".

### 2-2. Входные и выходные данные

#### Данные на вход

На вход программа получает сначала 2 натуральных числа `n` и `k`. Далее на вход программы идут n целых чисел - каждое будет храниться в списке `l` длины `n`. Т.к. ограничений на предел значений данных нет, то считаем, что `n` и `k` могут принимать значения от 1 и до максимального значения типа данных `long`.

|   | Тип                         | min значение     | max значение       | 
|---|-----------------------------|------------------|--------------------|
| n | Целое число (long)          | 1                | 2<sup>63</sup> - 1 |
| k | Целое число (long)          | 1                | 2<sup>63</sup> - 1 |

#### Данные на выход

По условию задачи, на выход мы получаем либо целое число `r`, либо сообщение "NO". Если результат программы - целое число, то его значение будет иметь диапазон от 0 до максимального значения типа данных `long`, т.к. его начальное значение будет 0, а возможные присваемые значения будут находится в диапазоне, в котором находится число `n`.

|   | Тип                         | min значение     | max значение       | 
|---|-----------------------------|------------------|--------------------|
| r | Целое число (long)          | 0                | 2<sup>63</sup> - 1 |

### 2-3. Выбор структуры данных

Сначала программа получает 2 натуральных числа, поэтому для их хранения можно выделить 2 переменных (`a`, `n`) типа `long`. Также для хранения всех последующих n целых чисел нужно отвести список целых чисел `l` типа `long[]` длины `n`.

|   | название переменной | Тип (в Java)| 
|---|---------------------|-------------|
| a | `a`                 | `long`      |
| n | `n`                 | `long`      | 
| l | `l`                 | `long[]`    | 

Вывод результата - это либо целое число, либо сообщение "NO". Для сообщения "NO" не требуется переменная, но для результата в виде индекса нужно создать переменную `r` типа `long`.
|   | название переменной | Тип (в Java)| 
|---|---------------------|-------------|
| r | `r`                 | `long`      |

### 2-4. Алгоритм

#### Алгоритм выполнения программы:

1. **Ввод данных и создание переменной для вывода результата**   
       Программа считывает два числа, обозначенные как `a` и `n`. Затем создаёт переменную `t` и присваивает ей значение, хранящееся в переменной `a`.

2. **Вход в цикл**   
       Начинается цикл со счётчиком for, который произойдёт n раз.

3. **Присвоение нового значения переменной t и вывод части результата**   
       Значение `t` заменяется на умноженный на 2 модуль остатка от деления `t` на 10. Затем происходит вывод значения `t` и пробел.

4. **Повторение цикла**   
       Далее цикл повторяется n раз.



### 2-5. Программа

```java
import java.io.PrintStream;
import java.util.Scanner;

public class Main {
    public static PrintStream out = System.out;
    public static Scanner in = new Scanner(System.in);
    public static void main(String[] args) {
        long n = in.nextLong(), k = in.nextLong();
        long [] l = new long[(int) n];

        long m = in.nextLong();
        l[0] = m;
        for (int i = 1 ; i < n ; i++) {
            l[i] = in.nextLong();
            if (l[i] > m)
                m = l[i];
        }

        long r = 0;
        for (int i = 0 ; i < n ; i++) {
            if (l[i] == m)
                k--;
            if (k == 0)
                r = i + 1;
            out.println(m + " " + k + " " + l[i]);

        }


        if (r == 0)
            out.print("NO");
        else
            out.print(r);

    }
}
```

Программа работает корректно на всем множестве решений. Далее разобраны примеры для различных вариантов вывода программы и значений ввода:

        ```

        1 2 3 4 5
        ```

    - **Output**:
        ```
        5
        ```

3. ***k > 1***
   
    - **Input**:
        ```
        5 3
        4 8 8 4 8
        ```

    - **Output**:
        ```
        5
        ```
