## Задание 1
### 1-1 Постановка задачи

> На вход программы подаются целое число a и натуральное число n.
> Необходимо вывести через пробел n чисел последовательности (начиная со второго элемента), заданной по следующим правилам:
> - Первый элемент последовательности - это число a.
> - Каждый последующий член последовательности - это умноженный на 2 модуль остатка предыдущего члена последовательности от деления на 10.

### 1-2. Входные и выходные данные

#### Данные на вход

На вход программа должна получать целое число a и натуральное число n. В условии нет ограничений на максимальное значение данных на вход чисел, поэтому будем считать, что число a может принимать любые значения, входящие в диапазон типа данных long, а число n может принимать значения от 1 до максимального значения типа данных long.

|   | Тип                | min значение     | max значение       | 
|---|--------------------|------------------|--------------------|
| a | Целое число (long) | -2<sup>63</sup>  | 2<sup>63</sup> - 1 |
| n | Целое число (long) | 1                | 2<sup>63</sup> - 1 |

#### Данные на выход

По условию задачи, на выход мы получаем n целых чисел, разделённых через пробел.

### 1-3. Выбор структуры данных

Программа получает целое и натуральное числа, поэтому для их хранения можно выделить 2 переменных (`a` и `n`) типа `long`.

Вывод результата - это целые числа. Каждое будет хранится перед выводом в 1 переменной (`t`) типа `long`.
|   | название переменной | Тип (в Java)| 
|---|---------------------|-------------|
| t | `t`                 | `long`      |

### 1-4. Алгоритм

#### Алгоритм выполнения программы:

1. **Ввод данных и создание переменной для вывода результата**   
       Программа считывает два числа, обозначенные как `a` и `n`. Затем создаёт переменную `t` и присваивает ей значение, хранящееся в переменной `a`.

2. **Вход в цикл**   
       Начинается цикл со счётчиком for, который произойдёт n раз.

3. **Присвоение нового значения переменной t и вывод части результата**   
       Значение `t` заменяется на умноженный на 2 модуль остатка от деления `t` на 10. Затем происходит вывод значения `t` и пробел.

4. **Повторение цикла**   
       Далее цикл повторяется n раз.



### 1-5. Программа

```java
import java.io.PrintStream;
import java.util.Scanner;

public class Main {
    public static PrintStream out = System.out;
    public static Scanner in = new Scanner(System.in);
    public static void main(String[] args) {
        long a = in.nextInt(), n = in.nextInt();
        long t = a;

        for (int i = 0 ; i < n ; i++) {
            t = Math.abs(t % 10 * 2);
            out.print(t + " ");
        }
    }
}
```

## Задание 2
### 2-1 Постановка задачи

> Дана послоедовательность из n чисел. Найти такой индекс, где максимум последовательности встречается k-ый раз. Если такого индекса нет - вывести "NO".

### 2-2. Входные и выходные данные

#### Данные на вход

На вход программа получает сначала 2 натуральных числа `n` и `k`. Далее на вход программы идут n целых чисел - каждое будет храниться в списке `l` длины `n`. Т.к. ограничений на предел значений данных нет, то считаем, что `n` и `k` могут принимать значения от 1 и до максимального значения типа данных `long`.

|   | Тип                         | min значение     | max значение       | 
|---|-----------------------------|------------------|--------------------|
| n | Целое число (long)          | 1                | 2<sup>63</sup> - 1 |
| k | Целое число (long)          | 1                | 2<sup>63</sup> - 1 |

#### Данные на выход

По условию задачи, на выход мы получаем либо целое число `r`, либо сообщение "NO". Если результат программы - целое число, то его значение будет иметь диапазон от 0 до максимального значения типа данных `long`, т.к. его начальное значение будет 0, а возможные присваемые значения будут находится в диапазоне, в котором находится число `n`.

|   | Тип                         | min значение     | max значение       | 
|---|-----------------------------|------------------|--------------------|
| r | Целое число (long)          | 0                | 2<sup>63</sup> - 1 |

### 2-3. Выбор структуры данных

Сначала программа получает 2 натуральных числа, поэтому для их хранения можно выделить 2 переменных (`a`, `n`) типа `long`. Также для хранения всех последующих n целых чисел нужно отвести список целых чисел `l` типа `long[]` длины `n`.

|   | название переменной | Тип (в Java)| 
|---|---------------------|-------------|
| a | `a`                 | `long`      |
| n | `n`                 | `long`      | 
| l | `l`                 | `long[]`    | 

Вывод результата - это либо целое число, либо сообщение "NO". Для сообщения "NO" не требуется переменная, но для результата в виде индекса нужно создать переменную `r` типа `long`.
|   | название переменной | Тип (в Java)| 
|---|---------------------|-------------|
| r | `r`                 | `long`      |

### 2-4. Алгоритм

#### Алгоритм выполнения программы:

1. **Ввод данных**   
       Программа считывает два числа, обозначенные как `n` и `k`. Также создаётся пустой список `l` длины n.

2. **Ввод последующих n чисел и определение максимума**   
       Создаётся целочисленная переменная `m`, принимающая значение первого из n чисел. Далее каждый введённый элемент записыватся в список `l`, а также сравнивается с переменной `m`. Если он больше `m`, то m принимает значение большего числа.

3. **Создание переменной, хранящей результат и определение её значения**   
       Создаётся переменная `r`, которой присваивается значение 0. Далее мы сравниваем каждый элемент списка, заполненного в предущем пункте, с максимальным. Найдя k-ый максимальный элемент, присваиваем новое значени `r`, равное порядковому номеру элемента (считаем от 1).

4. **Вывод результата**   
       Если `r` = 0 (т.е. его значение не изменилось), то выводим "NO". В ином случае выводим значение `r`.



### 2-5. Программа

```java
import java.io.PrintStream;
import java.util.Scanner;

public class Main {
    public static PrintStream out = System.out;
    public static Scanner in = new Scanner(System.in);
    public static void main(String[] args) {
        long n = in.nextLong(), k = in.nextLong();
        long [] l = new long[(int) n];

        long m = in.nextLong();
        l[0] = m;
        for (int i = 1 ; i < n ; i++) {
            l[i] = in.nextLong();
            if (l[i] > m)
                m = l[i];
        }

        long r = 0;
        for (int i = 0 ; i < n ; i++) {
            if (l[i] == m)
                k--;
            if (k == 0)
                r = i + 1;
            out.println(m + " " + k + " " + l[i]);

        }


        if (r == 0)
            out.print("NO");
        else
            out.print(r);

    }
}
```

## Задание 3
### 3-1 Постановка задачи

> Определите, является ли введенная в консоль строка правильной скобочной последовательностью (выведите “YES”/“NO”). В правильной скобочной последовательности каждая '(' имеет пару ')', а закрывающие скобки не встречаются раньше открывающих.

### 3-2. Входные и выходные данные

#### Данные на вход

На вход программа должна получать строку, состоящую из символов `(` и `)`.

#### Данные на выход

По условию задачи, на выход мы получаем строку "NO" или "YES".

### 3-3. Выбор структуры данных

Программа получает строку на вход, поэтому для их хранения можно выделить 1 переменную (`n`) типа `String`.

|   | название переменной | Тип (в Java)| 
|---|---------------------|-------------|
| n | `n`                 | `String`    | 

Вывод результата - это строки "NO" и "YES". Для их вывода не требуется создание переменных.

### 3-4. Алгоритм

#### Алгоритм выполнения программы:

1. **Ввод данных и создание необходимых переменных**   
       Программа считывает строку, состоящую из символов `(` и `)`. Далее создаются три целочисленных переменных: `l` и `r`, хранящие количество символов `(` и `)` соответственное, а также создаётся целочисленная перменная `res`, хранящее изначально значение 1.

2. **Анализ данной строки**   
       Каждый символ, входящий в данную строку увеличивает значение переменной `l` или `r`, после чего сравниваются эти переменные. Если `r` больше, чем `l`, то `res` меняет своё значение на 0 - т.е. данная строка не является "правильной", т.к. есть `)`, не имеющая пары.

3. **Вывод результата**
         Если `res` = 0  или значени переменных `l` и `r` не совпадают, то выводится "NO". Иначе выводится "YES".


### 3-5. Программа

```java
import java.io.PrintStream;
import java.util.Scanner;
public class Main {
    public static PrintStream out = System.out;
    public static Scanner in = new Scanner(System.in);
    public static void main(String[] args) {
        String n = in.nextLine();
        int l = 0, r = 0, res = 1;
        for (int i = 0 ; i < n.length() ; i++) {
            if (n.charAt(i) == '(')
                l++;
            else
                r++;
            if (r > l)
                res = 0;
        }

        if (res == 0 || r != l)
            out.print("NO");
        else
            out.print("YES");
    }
}
```

## Задание 5
### 5-1 Постановка задачи

> Дан массив из n чисел. Определить, есть ли в нем элемент, встречающийся в два раза реже, чем его значение. Выведите значение этого элемента или "NO", если такого элемента нет.

### 5-2. Входные и выходные данные

#### Данные на вход

На вход программа должна получать натуральное число `n`. Затем на программа должна получить на вход n целых чисел, которые будут записаны в массив `l` длины n. Ограничений на входящие данные в условии нет, но длина списка может быть объявлена только переменной типа `int`, поэтому `n` ограничен снизу 1 и максимальным допустимыми значениями типа данных `int` сверху. 

|   | Тип                | min значение     | max значение       | 
|---|--------------------|------------------|--------------------|
| n | Целое число (int)  | 1                | 2<sup>31</sup> - 1 |

#### Данные на выход

По условию задачи, на выход мы получаем либо строку "NO", либо значение искомого данного элемента. Для вывода необходимо использовать переменную `res`, которая принимает значения равыне значениям данным n элементам, поэтому ограничения у переменной `res` будут минимальным и максимальным возможным для типа данных `int`.

|     | Тип                | min значение     | max значение       | 
|-----|--------------------|------------------|--------------------|
| res | Целое число (int)  | -2<sup>31</sup>  | 2<sup>31</sup> - 1 |

### 5-3. Выбор структуры данных

Программа получает целое число, поэтому для его хранения можно выделить 1 переменную (`n`) типа `int`. Для хранения последующих n чисел можно использовать массив `l` типа `int[]` длиной n.

|   | название переменной | Тип (в Java)| 
|---|---------------------|-------------|
| n | `n`                 | `int`       | 
| l | `l`                 | `int[]`     | 


Вывод результата может быть целым числом, поэтому выделяем переменную `res` типа данных `int`.
|     | название переменной | Тип (в Java)| 
|-----|---------------------|-------------|
| res | `res`               | `int`      |

### 5-4. Алгоритм

#### Алгоритм выполнения программы:

1. **Ввод данных и создание необходимых в работе переменных**   
       Программа считывает одно натуральное число `n`. Затем записывает n целых чисел в массив `l`. Далее создаются 3 целочисленных переменных: `res`, `count` и `el` для хранения результата, количества элементов равным по значению исследуемому элементу и значение ииследуемого элемента соответственно.

2. **Анализ данной последовательности**   
       Исследуем каждый элемент: записываем его значение в переменную `el`, а также количество элементов с таким значением в списке в переменную `count`.

3. **Вывод результата**
         Если количество элементов с таким значением меньше значения элемента в 2 раза, то значение `res` заменяется на `el` - это и будет результатом. В случае, если таких элементов не было найдено, то `res` имеет значение 0 и результатом программы будет "NO".



### 5-5. Программа

```java
import java.io.PrintStream;
import java.util.Scanner;
public class Main {
    public static PrintStream out = System.out;
    public static Scanner in = new Scanner(System.in);
    public static void main(String[] args) {
        int n = in.nextInt();
        int [] l = new int[n];
        for (int i = 0 ; i < n ; i++)
            l[i] = in.nextInt();

        int res = 0, count, el;
        for (int i = 0 ; i < n ; i++) {
             el = l[i];
             count = 0;
             for (int j = 0 ; j < n ; j++) {
                 if (l[j] == el)
                     count++;
             }
             if (el == count * 2)
                 res = el;
        }

        if (res != 0)
            out.print(res);
        else
            out.print("NO");
    }
}
```
